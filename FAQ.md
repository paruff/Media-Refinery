# Frequently Asked Questions (FAQ)

**Q: How do I get started?**
A: See the Quickstart in the main README and the docs/ directory for details.

**Q: How do I run tests?**
A: Use `pytest` for unit/integration tests, `behave` for BDD, and see the Makefile for more.

**Q: Where do I put my media files for testing?**
A: Use the `sample_media/` directory.

**Q: How do I contribute?**
A: See CONTRIBUTING.md in docs/.

**Q: How do I report a security issue?**
A: Please open a private issue or contact the maintainers directly.

For more, see the full documentation in docs/.
